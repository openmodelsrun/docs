# Welcome to OpenModels

OpenModels is an open-source registry and comparison platform for LLM models and inference providers. It helps developers discover, compare, and integrate AI models across multiple providers with standardized data, real-time telemetry, and transparent pricing.

## Key Features

### Model Registry

A community-maintained, schema-validated registry of 60+ LLM models from vendors including OpenAI, Anthropic, Google, Meta, Mistral, DeepSeek, Microsoft, Alibaba, and more. Every model definition is machine-readable and version-controlled.

### Provider Comparison

Compare 30+ inference providers side-by-side — pricing, capabilities, context windows, and availability. Find the best provider for your use case with structured, up-to-date data.

### Real-Time Telemetry

Monitor provider health, latency, and reliability through live telemetry data. Make informed decisions about which providers to use based on actual performance metrics.

### Open Source

The entire registry is open source and community-driven. Contribute new models, providers, and mappings through pull requests with automated schema validation.
